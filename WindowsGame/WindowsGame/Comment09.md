## InputManager
---
```
 Windows API 에서 키인풋을 그동안 프로시저 통해서 받았던걸
 다른 함수 통해서 받아주고, 현재 키가 눌려져있는지, 눌렸는지 
 이런것들을 관리하는 클래스
```
