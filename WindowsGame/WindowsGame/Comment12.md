## 2. Vector2 보강 (게임수학)
```
 Vector2 a = {1,2}
 Vector2 b = {2,3}
 Vector3 c = a + b;

 이런 기본적인 연산자
```


```
 Vector2
 Z축이 없어서 게임수학이 많이쉽습니다.

 내적과 외적 (Z축을안쓰면 외적을 쓸일이 없습니다.)

 내적(Dot)
 A dot B
  내적의값 - A벡터를 B 벡터한테 투영시켰을때 걸치는 선의 길이 (실수형)

  여러가지 공식(수학자들이 많이 고안해서 만들어낸)
  
  1. A Dot B = |A| * |B| * cos(theta) 
  2. A Dot B = A.x *  B.x + A.y * B.y
   => 내적값 구하기
  
  cos(theta) => 
  코사인그래프로 인한 꼼수를 부릴수 있다.

  Tip 1. 
   - cos 그래프는
   - 0도 일때 1, 90도 일때 0, 180도 일때 -1 이다.

  Tip 2.
   - 내적으로는 180도가 넘는 각도가 존재할 수 없다.

  Tip 3.
   - other 벡터가 내 뒤에 있는지 확인하려면
   - 내적이 0보다 작으면 내 뒤에 있다. (90도가 넘는다는 소리)



 외적(Cross)
  - 사용안함.
  - 선이 두개있죠
  - 저 두개선과 공통으로 직각이 되는 선 (3D 에서만 존재)
```
