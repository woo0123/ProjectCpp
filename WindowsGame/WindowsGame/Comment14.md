## 오목 만들기

```
// 숙제 오목만들기
(백돌 흑돌 다 플레이어가 구현)
// 백돌은 쌍삼금지 쌍사금지

// 원 - 백돌
// 네모 - 흑돌
```

```
오목만들기 수도코드

 - 오목만들기가 여러분들이 콘솔에서 코딩했던느낌이랑 아주비슷해요.


 [어디서부터 시작하지?]
  - 눈에 보이는거 부터하세요
  어느정도 눈에보이면 그때부터 수도코드 짜도 안늦습니다.

  목표를 정해서 해볼게요.

  1. 오목판을 그려보자.
  2. 클릭한위치에 흑돌 / 백돌을 놓아보기
  3. 흑돌 다음에 백돌 두도록 하기.
  4. 승패판정
   - 수도코드
   돌을 둘때마다 
   그 돌과 연관된 가로세로 대각선 2개 에서 5개가 되었는지
    되었으면 승리.
  5. 쌍삼 쌍사 금지
  6. 버그수정
```
